DunGen

Dungeon Generator from Ed's Hobby Hovel

by Ed Allen

Plans for expansion blogged about this project at Ruminations of a Geek

What's new? Awesome new node editing dialog box. Added odd features to nodes. Expanded trap and potion descriptions, multiple treasure types in some hoards, and some multiple magic items (more in deeper levels), dungeon levels get one to three common monster types.
To manually save, copy the text of the room contents table and paste into another file, and right click the map to save the image as a PNG. Print looks best after moving the table using the button.
